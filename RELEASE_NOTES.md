# ApplyCairn 0.2.0 release notes

Initial private-review release for the official OpenAI plugin directory.

- Guided onboarding from the word `start`; no YAML or model API key.
- Resume-backed accomplishments, profile, and multi-lane search setup.
- Source-backed job discovery, deduplication, scoring, ranking, and tracking.
- Company and permitted public-contact research without LinkedIn scraping.
- Evidence-linked resumes, cover letters, review gates, and interview practice.
- Daily-update workflow with connector-aware status reconciliation.
- Explicit approval for sends, submissions, scheduling, edits, and deletion.
- Portable export, import, backup, and deletion of all user-owned data.
- Embedded onboarding, dashboard, jobs, accomplishments, tracker, daily briefing,
  contacts, approval, and data-control interfaces.
