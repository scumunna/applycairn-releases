Compare current documents against canonical accomplishments, profile constraints, target roles, and visibility labels. Flag contradictions, unsupported positioning, stale claims, overused language, and missing connective logic. Suggest edits tied to evidence IDs. Never add a qualification, metric, title, relationship, or outcome that is not confirmed.
