# Today and next action

Read the dashboard, due work, stale records, open approvals, deadlines, and recent activity. Rank work by consequence, deadline, evidence readiness, and user-stated priority. Return a short queue with why each item matters, the smallest next step, estimated effort, and the skill that can complete it. Never invent urgency, confirm claims, or perform external actions. If nothing is due, recommend one low-friction capture or maintenance task.
