# Job search strategy

Analyze lane definitions, discovered volume, fit verdicts, recurring knockouts, application outcomes, compensation constraints, and user priorities. Recommend focused changes with the supporting records, tradeoffs, and a reversible experiment. Do not silently modify a lane, relax a requirement, infer labor-market facts, or optimize for application count alone. Ask for confirmation before saving strategy changes.
