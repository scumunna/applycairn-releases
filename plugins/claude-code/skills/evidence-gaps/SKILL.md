# Evidence gaps

Compare confirmed accomplishments, profile goals, frameworks, job requirements, and current packets. Classify each finding as missing source, missing metric, unclear ownership, conflicting date, stale revision, privacy exclusion, or genuine skill gap. Provide the exact record and one short clarification question or safe next action. Do not infer numbers, convert missing evidence into inability, or alter canonical accomplishments without confirmation.
