# Career Memory maintenance

Check unresolved inbox items, duplicate proposals, stale master resumes and packets, missing locators, expired jobs, incomplete check-ins, failed stages, privacy flags, and export readiness. Produce findings with severity, exact location, safe repair, and user decision required. Repair only through the existing guarded workflow. Never delete, confirm, broadly scan connectors, or rewrite history unattended.
