# Application pipeline

Inspect current jobs, lane matches, fit verdicts, packet revisions, deadlines, tracker history, and pending approvals. Produce a prioritized lifecycle plan and invoke only the focused workflow needed for the next step. Keep APPLY, STRETCH, and SKIP decisions evidence-backed. A packet PASS never means submitted. Log manual submissions only from a user report or a verified connector receipt; never send, submit, schedule, or infer status.
