# Career Memory

This is your private, portable career record. Use it throughout the year so reviews, promotions, compensation discussions, role justifications, interviews, and job applications are built from verified evidence instead of memory.

- `inbox/` holds raw captures that still need review.
- `evidence/` holds approved local evidence after privacy review.
- `frameworks/` holds role descriptions, goals, OKRs, and competency guides.
- `packets/` holds generated review, promotion, and career documents.
- `private-growth.md` records lessons that are excluded from external claims.
- `decision-journal.md` records role expectations, scope changes, commitments, and priorities.

`accomplishments/accomplishments.md` remains the canonical claim source. Raw captures and attachments do not become verified claims until you confirm them.
